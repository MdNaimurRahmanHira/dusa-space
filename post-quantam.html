Quantum STEP Services

Security and encryption

The Quantum STEP Service enables organizations to determine the required security posture for a hybrid encryption solution that includes Post-Quantum Cryptography and Classical Encryption. The assessment includes a review of:

  • Business Events that trigger workflows and workload processing
  • Expressions associated with the underlying platform technologies
  • Enabling Experiences that map to user and group personas
  • Outcomes that are based upon metrics and success criteria

Types of Services

DUSA STEP Service deliverables include recommendations for the underlying platform technologies that can support Post-Quantum Cryptography. These technologies address the following types of services:

  • Cloud Providers : AWS, Azure, Google Cloud, plus
  • SaaS Providers : Salesforce, Workday, ServiceNow, plus
  • On-Premises : Applications, Infrastructure, Storage

And much more

In addition, value added services are also included that range from security providers to monitoring, logging, auditing, and detection services.
